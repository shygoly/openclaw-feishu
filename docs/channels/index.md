# Chat Channels

OpenClaw can talk to you over Feishu/Lark via the Gateway.
Text, media, and reactions are supported.

## Supported channels

- [Feishu](/channels/feishu) — 飞书/Lark enterprise messaging.

## Notes

- Group behavior varies; see [Groups](/concepts/groups).
- DM pairing and allowlists are enforced for safety; see [Security](/gateway/security).
- Troubleshooting: [Channel troubleshooting](/channels/troubleshooting).
- Model providers are documented separately; see [Model Providers](/providers/models).
